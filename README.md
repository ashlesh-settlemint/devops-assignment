## Start the application

1. Install dependencies

```
npm install
```

2. Check linting

```
npx nx run nft-bridge:lint
```

3. Run e2e tests

```
npx jest
```

4. Build the project

```
nx build nft-bridge
```

5. Run the application

```
node ./dist/apps/nft-bridge/main.js
```
